oh my zsh 配置

安装zsh

在bash中输入 zsh 即可进入 zsh，但是此时 通过 echo $SHELL​ 发现默认还是bash

​​

安装oh my zsh

需要先安装git

​sudo apt install git​

需要自行配置科学上网

​​

更改默认shell

​chsh -s $(which zsh)​

注销后再打开终端发现默认为 zsh

​​

安装ohmyzsh插件

安装插件z

Z是一个快捷路径跳转插件，输入z 通过 tab可以快捷选择曾经访问过的路径

​vim ~/.zshrc​

找到 plugins=(git)

修改为 plugins=(git z)​

​source ~/.zshrc​ 保存后重新加载zsh配置
