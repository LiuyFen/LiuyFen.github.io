Ubuntu搭建Django+VUE开发环境

最近打算写一个项目使用Django写API，VUE写前端，故打算在Ubuntu上搭建一套开发环境

‍

安装Mysql

  1. 验证mysql安装

    mysql -u root -p

    ​​

  2. 更改mysql root密码

    sudo mysql 进入mysql bash

    ​​

    use mysql;

    ALTER USER ‘root‘@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘新密码’ ;

    此时可以用新密码 通过 mysql -u root -p 进行登录

    ​​

‍

安装Python

  1. 安装pip

    当前环境无pip

    ​​

    sudo apt-get update

    sudo apt install python3-pip​

  2. 验证pip安装

    pip –version

    ​​

‍

安装Docker

  1. 验证docker安装

    docker –version

    docker-compose –version

    ​​

  2. 在Docker中启动多版本mysql

    mysql 8

    mysql 5

‍

安装Django

由于我是项目迁移，之前已经写好了部分代码，给关于Django编写API详细步骤未做记录，只记录迁移时遇到的问题

  1. No module named ‘corsheaders’

    ​pip install django-cors-headers​

  2. No module named ‘rest_framework’

    ​pip install djangorestframework​

  3. Did you install mysqlclient?

    ​pip3 install pymysql​

    在settings.py 同级目录的 init.py 添加如下代码

  4. No module named ‘rest_framework_simplejwt’

    ​pip install djangorestframework-simplejwt -i https://pypi.tuna.tsinghua.edu.cn/simple​

  5. No module named ‘pandas’

    ​pip install pandas​

  6. django.db.utils.OperationalError: (1049, “Unknown database xx)

    创建对应数据库

    ​​

  7. 应用迁移

    ​​

    ​python manage.py makemigrations​

    ​python manage.py migrate​

    ​​

  8. 启动Django项目

    ​python3 manage.py runserver 8001​

    ​​

‍

安装VUE

  1. 验证VUE安装

    ​vue -V​

    ​​

  2. 启动VUE项目

    权限报错解决

    ​​

    ​chmod 777 node_modules/.bin/vue-cli-service​

    ​​
